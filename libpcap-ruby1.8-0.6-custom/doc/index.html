Ruby/Pcap extension library

Ruby interface to LBL libpcap (Packet Capture library).

Modules

Following classes are defined under module Pcap. When you use these classes, you need to include module Pcap or refer them like Pcap::Capture.

  • Pcaplet
  • Packet
    • IPPacket

      • UDPPacket
      • ICMPPacket
  • IPAddress
  • Dumper
  • Filter
